What is Reveal.js?

  • Framework for HTML-based presentations
  • Supports Markdown
  • Supports themes and transitions
  • Vertical and horizontal navigation

Navigation Structure

  • Horizontal slides: Left / Right arrows
  • Vertical slides: Up / Down arrows
  • Keyboard and touch supported
